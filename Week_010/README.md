# 知识点

一维数据结构：数组，链表，栈，队列，双端队列，映射，集合【映射和集合都分为 hash 与 Tree】

二维数据结构：树，图，二叉搜索树（AVL，红黑），堆（多种实现，手撕一般为二叉堆【记模板】），并查集【记模板】，字典树【记模板】

特殊数据结构：位运算【N皇后，判重】，布隆过滤器【二进制+哈希函数，全为1不一定存在，存在0一定不存在】，LRU Cache【哈希+双向链表，O(1)查找，插入，删除】
